<h2>
  System to identify anomalies in vibration signals for local damage detection in machines
</h2>
The thesis concerns the development of a tool for identifying informative frequency bands in vibration signals of rotating machinery for fault detection purposes. A model of the signal and time–frequency analysis using the short-time Fourier transform and spectrogram are presented. Selected statistical selectors are compared using simulated and laboratory data. A web application tool has been developed to assist in the analysis of machine vibration signals.
